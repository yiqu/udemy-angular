Practice 11 - Http in Angular
Requirments:
  1. Send CRUD request in Angular.
  2. Use HttpClient to make GET, POST, PUT, and DELETE requests.
  3. Sending a request with Angular can be done with extra options:
    • headers?: HttpHeaders
    • observe: 'response | body | etc..';
    • params?: HttpParams
    • reportProgress?: boolean;
    • responseType?: 'json | text | etc..';
    • withCredentials?: boolean;
